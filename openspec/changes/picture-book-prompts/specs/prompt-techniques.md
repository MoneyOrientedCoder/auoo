# AI 绘本提示词技巧规格

## 核心方法论

### 1. 连续性语境管理 (Context Management)

**问题**：单独调用提示词，每个生成任务是孤立的。

**解决方案**：滚雪球式多步生成
- 生成第1页时：引用原图作为风格基础
- 生成第2页时：引用原图 + 第1页
- 生成第3页时：引用原图 + 第1页 + 第2页
- ...

**效果**：保证星星形状、人物特征、线条质感始终一致

### 2. 语义级提取与风格化应用

**分离处理**：
- **风格编码 (Style Encoding)**：蓝色墨水、钢笔涂鸦、纸张纹理、飞溅质感
- **语义编码 (Semantic Encoding)**：特定阶梯形状、星形画法、月亮、小车

**映射方法**：将编码映射到新构图，保持风格+语义一致

### 3. 隐藏语境指令 (Hidden Context Instructions)

不能简单用文字表达的核心指令：

| 指令类型 | 示例 |
|----------|------|
| 风格锁定 | "锁定且必须使用原图的特定蓝色墨水纹理。禁止使用平滑矢量线条" |
| 元素一致性 | "星星必须完美复制原图中那个特定的、不规则的星形画法" |
| 继承与重组 | "第2页是阶梯的更高视角；第3页是滑梯的特写视角" |
| 人物特征 | "保留原图中高发髻特征，作为标志性符号" |

## 提示词模板

### 基础模板
```
[场景描述], inspired by [原图], refined into [优化方向].
The [核心元素] must be preserved from [参考图].
Style: [风格描述], [质感描述].
[人物/动作描述].
```

### 连续性模板
```
[新场景描述], looking [视角方向] along the [结构] from image_X.png.
[角色] with [特征 from 原图] is [动作].
The [元素] (identical to those in image_X.png) [行为].
Style remains [风格锁定].
```

## 五页生成示例（ Starlight Amusement Park）

| 页面 | 核心技巧 |
|------|----------|
| 第1页 | 继承原图风格，建立基石 |
| 第2页 | 引用第1页 + 原图，改变视角 |
| 第3页 | 动态风格化，保持元素一致 |
| 第4页 | 精准复现特定物体 |
| 第5页 | 状态改变（fading）+ 引入新颜色 |

## 关键原则

1. **禁止孤立生成**：每张新图必须引用之前的图
2. **强调 identical**：关键元素必须标注"identical to"
3. **质感锁定**：用具体描述锁定视觉风格
4. **状态渐进**：允许改变（动态、 fading），但需明确标注
